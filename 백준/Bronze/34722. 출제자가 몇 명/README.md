### 문제 설명

<p>이번에 처음으로 동국대학교 대회가 백준(Baekjoon Online Judge)에서 열린다!</p>

<p>백준에서 내는 만큼 엄격한 기준을 만족해야 출제가 가능하다. 이번에 대회를 총괄하는 강민이는 누가 출제할 수 있는지 확인해보려고 한다. 출제 자격이 생기는 조건은 다음과 같다.</p>

<ul>
<li>BOJ에서 해결한 문제 수가 $1000$ 이상이다.</li>
<li>Codeforces 레이팅이 $1600$ 이상이다.</li>
<li>Atcoder 레이팅이 $1500$ 이상이다.</li>
<li>ICPC 리저널 본선에서 $30$등 이내를 기록했다.</li>
</ul>

<p>위의 조건 중 하나라도 만족하면 출제 자격이 생긴다. 과연 몇 명이 출제할 수 있을까?</p>

### 입력 

 <p>첫 번째 줄에 사람의 수 $N(1 \le N \le 100)$이 주어진다.</p>

<p>다음 $N$개의 줄에 네 정수 $s_i$, $c_i$, $a_i$, $r_i$가 공백으로 구분되어 주어진다. 각 값의 범위는 아래와 같다.</p>

<p>BOJ 해결한 문제 수 $s_i$: $0 \le s_i \le 100{,}000$</p>

<p>Codeforces 레이팅 $c_i$: $0 \le c_i \le 4000$</p>

<p>AtCoder 레이팅 $a_i$: $0 \le a_i \le 4000$</p>

<p>ICPC 리저널 본선 등수 $r_i$: $r_i=-1$ 또는 $1 \le r_i \le 200$ (본선에 나가지 못했다면 $-1$이 주어진다.)</p>

<p>모든 값은 정수로 주어진다.</p>

### 출력 

 <p>출제 자격이 있는 사람의 수를 출력한다.</p>
